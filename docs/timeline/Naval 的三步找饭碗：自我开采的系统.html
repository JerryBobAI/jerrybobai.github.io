自我开采

Naval's 3 Steps: 找饭碗，不是选行业，是选赛道 + 打法

先把虚头巴脑撕掉，直看三步。你的“这碗饭”，是你用专长、杠杆和复利，亲手熬出来的。

三步找饭碗

1

发掘天赋

Uncover Unique Skill

发掘被忽视的天赋——那些你做得异常轻松、别人做得异常吃力的东西。不靠证书，靠长期可重复的优势。

优势壁垒： 轻松完成，他人吃力。
2

验证真需

Validate Market Fit

验证市场真需——用最小样本接单、收钱、复盘，把“自我感觉良好”打回“用户愿意付费”的原形。

最小闭环： 样本测试，现金流反馈。
3

搭建系统

Build Compounding System

搭建能复利的系统——流程化、产品化、资产化，让今天的努力能给明天持续供能，而不是一次性燃烧。

持续供能： 资产化，避免一次性燃烧。

财富公式

财富 = 专长 x 杠杆 x 复利

专长 (Specific Knowledge)

独特、稀缺、可被识别。不是“会用工具”，而是“能定义问题”。

杠杆 (Leverage)

代码、媒体、资本三类放大器。选你能掌控、不求人也能跑的那一类。

复利 (Compounding)

时间的同盟军。靠系统与信誉，把一次成功变成惯性成功。

今日三刀

切割虚荣

列出3个你轻松他人吃力的技能，删去带来虚荣无现金流的那一个。

七日收钱

设计一个能在7天收钱的最小服务/产品，价格低但标准硬。

流程固化

把交付流程固化为清单与模板，让第二个客户的边际成本下降。

点火。

别再把“找饭碗”当选行业，现在就行动：
在你天然优势里找能被杠杆放大的动作，用系统让它越跑越快。
