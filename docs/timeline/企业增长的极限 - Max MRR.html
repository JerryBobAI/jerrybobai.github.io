Max MRR

增长的“天花板”

这项指标，用来预测企业因客户流失与新收入增长而注定遇到的营收天花板。它是一把切开自我麻醉的利刃，让你直观地看到未来会撞上的极限。

MAX MRR
N / C

N = 每月新增 MRR

C = 月流失率

最大 MRR = 每月新增 MRR / 月流失率

一个冷酷的真相。

假设每月新增 1000美元，流失率 5%，那么你的最大 MRR 就是 2万美元。 无论你怎么努力，最终都只能在这个天花板下徘徊。

增长，为何总在
半途而废？

创业初期增长很快，但很快就慢下来。不是市场小，而是客户一多，流失的力量成倍放大，新增再多也填不满流失的黑洞。

真正的增长杠杆。

真正的增长，不在于拼命拉新，而在于死磕留存。Max MRR是领先指标，它能提前预警好消息和坏消息。

NRR > 100%

(净收入留存)

如果公司NRR超过100%，公司就能自我增长，天花板消失。
