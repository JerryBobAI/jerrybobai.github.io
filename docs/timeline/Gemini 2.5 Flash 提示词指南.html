提示词指南

最快最高效的多模态模型
MULTIMODAL ARCHITECTURE

核心原则

PRINCIPLES

描述场景，而不是列举关键词。

模型的核心优势在于深度语言理解，叙述性的描述段落几乎总是比简单的关键词列表产生更好、更连贯的图像。

1

真实摄影场景

PHOTOREALISTIC

像摄影师一样思考，详细描述相机角度、镜头类型、光照和细节。

模板:

真实的 [拍摄类型] + [主体] + [环境] + [光照描述] + [相机/镜头细节]

2

风格化插图贴纸

STYLIZED STICKERS

明确指定艺术风格，记得要求白色背景用于项目素材。

模板:

[风格] 贴纸 + [主体特征] + [色彩搭配] + [线条和阴影风格] + 白色背景

3

精准文字渲染

TEXT RENDERING

Gemini擅长文字渲染，清楚描述文字内容、字体风格和整体设计。

模板:

创建 [图像类型] + 文字"[具体文本]" + [字体风格] + [设计描述] + [配色方案]

4

产品样机摄影

PRODUCT MOCKUPS

创建专业的产品照片，适用于电商、广告和品牌推广。

模板:

高分辨率产品照 + [产品描述] + [背景表面] + [灯光设置] + [相机角度]

5

极简负空间设计

NEGATIVE SPACE

适合创建网站背景、演示文稿，为文字叠加留出空间。

模板:

极简构图 + 单一 [主体] + [位置] + 大量空白 [颜色/画布] + 柔和光照

6

连续艺术面板

SEQUENTIAL ART

创建引人入胜的视觉叙事，适用于故事板、漫画条或任何连续艺术形式。

模板:

单个漫画面板 + [艺术风格] + [前景角色动作] + [背景设定] + [对话框文字]
