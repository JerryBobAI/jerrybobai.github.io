大认知模型

1. 结果导向

Outcome-Oriented Thinking

认知升级：聚焦核心杠杆点

过程导向：关注“我该怎么做”，追求完美过程，努力但低效。

结果导向：倒推实现目标所需的核心条件，将精力聚焦于对结果影响最大的关键杠杆点。

可学到的：清晰定义目标，避免无效的“努力”。

2. 价值交换

Value Exchange Mastery

突破工时，扩大价值边界

低价值：仅按岗位描述完成工作，被动等待任务。

高价值：主动创造价值，解决更高层级领导的问题，扩大价值边界。

可学到的：职场晋升不是看“工时”，而是看“解决了多大的问题”。你的价值边界越广，越不可替代。

3. 跨界迁移

Applied Methodology

内化底层逻辑为直觉

知识点：学会一套方法论（如万达的营销方法论）。

应用：将底层逻辑跨界应用，将知识内化为直觉（如将 ROI 模型完整应用于微商）。

可学到的：真正的学习是“跨界迁移”。商业的底层逻辑在不同行业是相通的。

4. 持续迭代

Continuous Learning

构筑商业壁垒（护城河）

执行者：盲目跟风，没有长远规划，市场消失时被淘汰。

战略家：意识到护城河的重要性，进行饱和攻击，并通过高层次学习确保认知与时代同步。

可学到的：在存量竞争时代，只有壁垒才能带来持续的利润。新的尝试必须用可持续性、天花板等数据指标来衡量。
