开学前，这场谈话至关重要

YOUR FIRST CONVERSATION BEFORE PRIMARY SCHOOL

01

心理准备

✓

娃要升级：从“游乐场”到“训练营”

上午连上3-4节课，课间休息仅10分钟，需快速完成喝水、如厕和换书本。

✗

别指望老师像幼儿园一样事事关照。

提前给娃打预防针：课间先办“正事”，再玩！

✓

家长要降低关注期待：“没消息就是好消息”

小学一个老师面对50+学生，无法做到只关注你家娃。信任老师，娃适应更快。

02

上厕所的事

✓

重中之重：带纸！

养成随身带小包纸巾的习惯，放进书包侧兜或口袋。

✓

课间黄金10分钟：撒尿优先

不要贪玩！如果上课憋不住，要勇敢、小声地告诉老师：“老师，我实在憋不住了！”

✗

千万别硬憋！

湿裤子会带来尴尬和不适，容易给孩子留下心理阴影。

03

保护自己

✗

危险地带：严禁追逐打闹！

走廊、楼梯口、人群密集处，摔跤的后果不是闹着玩的。

✗

警惕“小暗器”：

看到同学拿着铅笔、尺子、剪刀当“武器”，要赶紧躲开，小心误伤！

✓

勇敢说“不”！

同学拉你玩危险游戏（如爬高、扔石头）时，一定要大声拒绝。

04

家长别说空话

✓

上课时：指令明确

“嘴巴像拉链拉上，有提问才开口；管好手，不玩文具；眼睛看老师和黑板。”

✓

见到老师：主动喊“老师好！”

嘴甜有礼貌的孩子，老师都喜欢。这是建立良好关系的第一步。

05

需要培养的习惯

阅读是受益终身的能力。每天精读15分钟，日积月累，丰富阅读量。阅读力是所有学科学习的基础，尤其是一二年级的孩子，越早熟悉，越早打好基础。

✗

纯粹读名著

对小娃太难，常常导致他们上课打瞌睡或溜号，效果不佳。

✓

线上精读课

专业老师带领，每天仅需15分钟，教授识字和理解故事的方法，培养扎实的阅读能力。

坚持的力量

坚持小半年后，孩子的识字量暴增，阅读速度显著提升，写看图写话也更加得心应手，常受老师表扬。
