动态网页生成助手

提示词用法详解

理解并掌握强大的动态网页生成提示词，轻松定制多样化的视觉风格与布局。

提示词概览

该提示词引导AI助手，根据用户提供的内容，结合预设的视觉风格和布局规则，生成一个定制化的中文动态网页。

核心运作逻辑：

  1. 输入理解：分析用户文本或链接，识别内容与潜在偏好。
  2. 风格决策：从12种预设风格（如 科技渐变 (默认), 吉卜力动画, 暗黑科技 等）中选择，优先用户指定。
  3. 布局决策：从7种预设布局（如 Hybrid Layout, Bento Grid 等）中选择，有特定风格的默认布局，优先用户指定。
  4. 决策输出：明确告知用户最终选择的风格和布局。
  5. 网页生成：基于选择和详细规范，输出HTML, CSS (TailwindCSS) 及JS代码。

可选风格详解

AI助手可根据以下12种风格生成网页，每种风格都有其独特的色彩、背景和视觉特点。

科技渐变 Tech Gradient (默认)

白色背景，黑色文字，蓝/紫/橘红等鲜艳科技渐变高亮色。现代、动感、有活力。

吉卜力动画 Ghibli Animation

柔和自然色调，手绘纹理，温暖浅色。文字和谐，图标手绘感。营造治愈、温馨的故事感。

暗黑科技 Dark Tech

深邃暗色背景，高对比亮色文字/UI，高饱和科技感高亮色。营造未来、赛博朋克或高端神秘感。

插画叙事 Illustrative Storytelling

背景与色彩服务于插画故事氛围，文字/UI从插画色盘提取或用中性色。强调视觉故事与情感连接。

孟菲斯设计 Memphis Design

大胆明快纯色块，几何图形，高对比文字。活泼、有趣、复古未来感。

极简侘寂 Minimalist Wabi-Sabi

素雅自然大地色系，可带肌理感，文字柔和对比。宁静、质朴、内敛，强调留白与自然美。

新国风工笔画 New Chinese Gongbi

素雅宣纸色/淡雅传统色背景，沉稳墨色/赭石色文字，古典低饱和高亮。雅致、清逸、东方美学。

质感设计 Material Design

白/浅灰基底，Material Design调色板主色与强调色，通过阴影体现层级。物理隐喻、动态反馈。

谷歌发布会 Google Keynote

极简白/浅灰背景，大量留白，Google品牌色高亮，Material Design原则。简洁、现代、专业、信息聚焦。

小米发布会 Xiaomi Keynote

常见深邃暗色背景，品牌橙色核心强调，清晰参数呈现。科技感强，信息详实。

飞书网站 Feishu Website

纯净白/浅灰背景，飞书品牌蓝色主导高亮，清晰层级与功能导向。现代、简洁、专业SaaS产品形象。

小红书 Xiaohongshu

高质量图/短视频为主体，色彩明亮清新饱和度偏高，卡片式布局，大量标签化。生活美学，社区连接。

可选布局详解

根据内容特性和所选风格，AI助手会匹配或允许用户指定以下布局方式。

Hybrid Layout (默认通用)

混合布局，灵活组合不同模块，适应多样化内容呈现。

Structured Information Layout

结构化信息布局，强调清晰信息层级和逻辑流。 (谷歌发布会风格默认)

Balanced Info-Graphic Layout

均衡信息图表布局，平衡大量图文信息与视觉吸引力。(小米发布会风格默认)

Section-based Layout

分区块布局，将内容组织在清晰的横向区块中。(飞书网站风格默认)

Card Layout

卡片布局，内容以独立卡片形式展示，适合信息流。(小红书风格默认)

Bento Grid

便当网格布局，将不同大小和类型的内容块组合在网格中，富有设计感。

Pinterest-style Grid

瀑布流网格布局，高度不一的卡片多列展示，适合图片密集型内容。

  • check_circle_outline网页需响应式兼容1920px及以上宽度。
  • check_circle_outline中英文混用，中文大字体粗体，英文小字点缀。
  • check_circle_outline简洁勾线图形化作为数据可视化或配图元素（随风格调整）。
  • check_circle_outline运用高亮色自身透明度渐变制造视觉效果，不同高亮色不互相渐变。
  • check_circle_outline数据可引用在线图表组件，样式需与主题一致。
  • check_circle_outline使用HTML5、TailwindCSS 3.0+ (CDN) 和必要JavaScript。
  • check_circle_outline使用专业图标库 (Font Awesome/Material Icons via CDN)。
  • check_circle_outline避免使用emoji作为主要图标。
  • check_circle_outline不要省略内容要点。
  • check_circle_outline过程输出也采用简体中文。
