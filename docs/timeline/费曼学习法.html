Step 1: Build Connections

海量阅读：知识的扎根

只有足够的输入，才能在思维中形成丰富的连接和想象力。这是费曼类比天赋的基石。

  • 建立知识的"图景"（Mental Map）。
  • 培养跨学科的类比想象力。
  • 避免知识的孤立化。

2

第二遍：懂得过程

Step 2: Deepen Memory

二次输入：由了解到懂得

通过拍照OCR或手抄关键部分，将知识从被动接收转化为主动整理和消化。

  • 手抄：加深肌肉记忆和专注力。
  • OCR/整理：形成精炼、可检索的个人笔记。
  • 核心目的：将知识刻入潜意识。

Step 3: Teach to Learn

公开分享：顿悟的终点

这是最重要的阶段。当你能用自己的语言、通过白话类比向陌生人讲解深刻道理时，即为融会贯通、顿悟。

  • 媒介：博客、社交媒体、演讲等。
  • 目标：用简单语言重组并传达复杂概念。
  • 效果：暴露知识盲区，倒逼理解。
