五大底层板块：AI时代“教什么”

1. How to See the World

学「如何看世界」

模型化的通识，而不是碎片知识。

  • 系统思维：因果链、反馈环、延迟效应、意外后果 —— 理解气候、经济、网络舆论。
  • 概率和不确定性：风险、期望值、贝叶斯直觉，知道“世界没有标准答案”。
  • 演化视角：从生物演化到技术演化、商业演化，理解“为什么会变成今天这样”。

2. Thinking Methodology

学「如何思考」

从“解题技巧”升级为“思维工法”。

  • 问题拆解能力：把一个模糊问题拆成可操作的小问题（例如：“我想做一个游戏”）。
  • 多方案思考：同一个问题，至少想到 3 种完全不同的解法，再对比选最优。
  • 判断与取舍：识别信息里的偏见、噪音，知道“什么可以信”和“信到什么程度”。

本质上是：从“我会做题”到“我会做选择”。

3. Human-AI Collaboration

学「如何和AI协作」

AI素养：现代版“读写算”基础学科。

  • 会问：把目标、限制条件、风格要求说清楚，而不是一句“帮我写作业”。
  • 会改：把 AI 的输出当“草稿”和“材料库”，学会自己删减、重组、提炼。
  • 会搭流程：用 AI 做资料搜集 → 结构设计 → 初稿 → 视觉生成，完成一个真正的作品。

4. Project-Based Learning

学「做事」

真实项目，组合技能，创造价值。

  • 项目制学习（PBL）：用知识 + AI 图像工具，制作“未来城市生存指南”或“想象中的物种图鉴”。
  • 团队协作与角色分工：学习时间管理、冲突解决、妥协、说服别人、抗压和情商。
  • 实践作品：一个小游戏、一本小绘本、一段解说视频、一个对别人有用的东西。

真正有竞争力的，是能将兴趣与AI组合起来的孩子。

5. Becoming Human

学「成为人」

AI越聪明，人类越需要那些“AI没法替你活的部分”。

价值判断与责任感：能区分“能不能做”和“该不该做”。
情绪管理与韧性：接受自己会输、会错，学会在焦虑里行动，而不是躺平。
审美与品味：在AI生成的无限结果中，判断“哪个是真的好？适合这个场景？”。

这些曾被视为“软性”的特质，在AI时代是核心竞争力。

传统学科：不是砍掉，是「重写大纲」

都还要学，但要围绕“模型 + 方法 + 表达”重构教学目标。

6. 数学 / MATH

从“算题”变成“理解世界底层结构”。

  • 聚焦：函数和变化趋势、概率与统计（判断风险）、基本逻辑结构（充分条件/必要条件）。
  • 理念：大量计算交给 AI，人学判断风险和看懂数据。

7. 语文 / LANGUAGE

从“应试作文”变成“表达与叙事能力”。

  • 聚焦：讲一个人听得进去的故事；把复杂的观点讲清楚；与不同的人沟通。
  • 理念：课文的选择从“背诵型”转为“思考型”，关注立场与看不到的部分。

8. 理化生 / SCIENCE

从“背知识点”变成“理解方法 + 探究习惯”。

  • 聚焦：怎样提出假设？怎样设计验证？怎样解释“和预期不一样的结果”？
  • 理念：科学不是结论，是一种不断修正自己认知的方式。

9. 历史 / HISTORY

从“记年代”变成“理解人类如何做选择”。

  • 聚焦：挑选重大节点，帮孩子看见不同决策 → 不同后果。
  • 理念：让孩子学会：“如果我是当事人，在当时信息条件下，我会怎么选？”。

小步快跑：立即行动建议

如果你是家长 / 教师 / 决策者：从日常开始改变。

1. 减少纯“抄资料式作业”

允许孩子用 AI 查资料，但要求他们自己写“总结 + 判断 + 反思”。

2. 所有作业默认有一个“为什么”问题

不是只交答案，而是：你为什么这样做？还有别的做法吗？如果条件变了，你的答案会怎么变？

3. 每学期至少做 1–2 个真正“对别人有用”的项目

小册子、小网站、小展览、小科普视频都可以。

4. 日常对话里，多问孩子“你怎么看”“你会怎么选”

少问一点“你背到哪一课了”，将重点从知识搬运转向观点形成。
