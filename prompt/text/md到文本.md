将以下 Markdown 文本转换为适合长推文或微博帖子的纯文本格式。

严格遵守以下规则：
1. 保留核心内容。
2. 仅输出纯文本。移除所有 Markdown 语法（如 `*`、`_`、`#` 等），但保留下面的 blockquote 规则。
3. **链接处理：** 默认情况下，*删除所有超链接*（保留链接文本并删除 URL）。
   * **例外：** 保留对理解上下文至关重要的链接，例如文章来源或其他您认为对读者理解至关重要的链接。
   * 将这些保留的链接格式化为：`链接文本( https://example.com )`
4. 在标题和段落之间添加空行，以确保可读性。
5. 将 Markdown 标题（例如，`## 我的标题`）转换为纯文本（例如，`我的标题`）。
6. 将无序列表格式化如下：
   * 级别 1：`- 列表项`
   * 二级：` * 子项`（缩进 2 个空格）
7. 通过保留其原始编号或字母来格式化有序列表（例如，`1.`、`a.`、`I.`、`1)`）。
   * 嵌套有序列表用空格缩进（例如，` 1. 子项`）。
8. 使用 `> Quoted text` 格式保留引用块。
9. 删除所有非内容、元数据或引用标签（例如，``，``）。
