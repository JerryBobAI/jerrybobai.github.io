写作原则（9条核心要求）：
✓ 使用简单语言，句子要简短、平实
✓ 避免AI腔：别用"深入探讨""释放潜能""颠覆性变革"
✓ 直接简洁，删掉多余的词
✓ 保持自然语气，像真人说话，可以用"而且""但是"开头
✓ 不用营销话术，不夸张，不制造hype
✓ 保持诚实，不假装亲切，不做无法兑现的承诺
✓ 简化语法，口语化表达更有人味也没问题
✓ 去掉冗余内容，避免不必要的形容词或填充词
✓ 注重清晰度，让读者容易理解

→ 原文：[粘贴你要改写的文本]

硬性限制（6条严禁）：
✗ 不得使用破折号
✗ 不得使用列表或"X，而且还有Y"句式
✗ 除非原文需要，否则不用冒号
✗ 避免设问句，如"你有没有想过……？"
✗ 句首句尾不用"基本上""显然""有趣的是"
✗ 不用虚假互动语句，如"我们一起来看看""和我一起踏上旅程""准备好迎接改变吧"

最重要的：
让文字读起来真实、自然，不像机器人写的，也不带推销感。如果有不清楚的地方，先提问澄清后再开始。
